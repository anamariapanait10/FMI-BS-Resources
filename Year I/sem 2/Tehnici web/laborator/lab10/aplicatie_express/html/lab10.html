Tema 10.1

Se considera documentul html p1.html. Implementati o aplicatie server folosind Node.js si Express astfel incat la submit pe formularul dat (cu metoda get), sa se calculeze media aritmetica a notelor trimise si sa se afiseze,
in interiorul paginii cu formularul, in iframe, intr-un paragraf, textul "Studentul X are media M".

Tema 10.2

Se considera documentul html p2.html.
La server se va defini si vectorul de obiecte, dictionar = [{cuvant:"carte", engleza:"book", franceza:"livre"}, {cuvant:"floare", engleza:"flower", franceza:"fleur"},{cuvant:"tablou", engleza:"picture", franceza:"photo"}, {cuvant:"film", engleza:"movie", franceza:"film"}]. Scrieti aplicatia server astfel incat dupa introducerea in inputul de tip text a unui cuvant si selectarea unei optiuni, la click pe butonul de submit, cuvantul si optiunea selectata se vor trimite catre server (cu metoda post) iar in pagina se va afisa traducerea cuvantului in limba selectata daca acesta se gaseste in dictionar si un mesaj in cazul in care nu exista cuvantul in dictionar.

Tema 10.3

Se considera documentul html p3.html si fisierul persoane.json.
Implementati o aplicatie server folosind Node.js si Express astfel incat la submit pe formularul dat (cu metoda get/post), pentru cheia oferita sa se afiseze ca lista ordonata,
in interiorul paginii cu formularul, in iframe, toate numele complete asociate prenumelui/numelui (conform select) respectiv.

Tema 10.4

Documentul HTML p4.html contine doua elemente input de tip text avand etichetele "Email" si "Subiect", un element de tip textarea cu eticheta "Mesaj" si un buton cu textul "Trimite mail".
Implementati o aplicatie server folosind Node.js si Express astfel incat la submiterea formei cu metoda POST sa se trimita un mail la adresa introdusa in campul "Email" si avand ca subiect
textul introdus in campul "Subiect" iar continutul mesajului sa fie textul introdus in campul "Mesaj" (folositi modulul nodemailer).
